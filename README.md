# SimpleCheckInApp

这是一个简单的 Android 打卡 App。

它可以按不同主题记录每天是否打卡，数据保存在本地。下方会用热力图显示打卡记录，也支持补打卡、主题重命名、删除主题、导入数据和导出 CSV。

## 运行

用 Android Studio 打开 `SimpleCheckInApp` 目录，等 Gradle Sync 完成后运行 `app`。

调试 APK 会生成在：

```text
SimpleCheckInApp/app/build/outputs/apk/debug/app-debug.apk
```
